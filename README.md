# DreamBees

DreamBees is a high-performance, interactive platform designed for content generation, 3D visualization, and community showcasing. Built with a modern tech stack, it provides a seamless experience for creators to build, share, and explore digital assets.

## 🚀 Key Features

- **Interactive 3D Studio**: Advanced visualization using React Three Fiber and Three.js.
- **Content Generation**: Integrated tools for creating and editing mockups and digital assets.
- **Social Feed & Showcase**: A dynamic platform for users to discover and showcase their creations.
- **Secure Authentication**: Robust user management powered by Firebase Auth.
- **Fluid UI/UX**: Smooth animations and transitions using Framer Motion and Lenis scroll.
- **Responsive Design**: Fully optimized for both desktop and mobile experiences.

## 🛠 Tech Stack

- **Frontend**: [React](https://reactjs.org/), [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/), [Framer Motion](https://www.framer.com/motion/)
- **3D Hub**: [Three.js](https://threejs.org/), [React Three Fiber](https://docs.pmnd.rs/react-three-fiber)
- **Backend & Persistence**: [Firebase](https://firebase.google.com/) (Auth, Firestore, Storage)
- **Analytics**: Google Analytics 4
- **HLS Video**: [hls.js](https://github.com/video-dev/hls.js/)

## 🏁 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd DreamBeesv11
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add your Firebase and API configurations (see `.env.example`).

### Development

Run the development server:
```bash
npm run dev
```

### Production

Build the application for production:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```
